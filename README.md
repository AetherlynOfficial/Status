# Aetherlyn Hosting Status

Landing page for [Aetherlyn Hosting Status](https://github.com/AetherlynOfficial/Status) — Minecraft hosting done right.

## Deploy to Vercel

1. Push this repo to GitHub (`AetherlynOfficial/Website`).
2. In [Vercel](https://vercel.com), import the repository.
3. Use the default settings (static site — no build command needed).
4. Deploy.

The site is a static HTML/CSS/JS project. Vercel serves `index.html` from the root automatically.

## Local preview

Open `index.html` in a browser, or run a simple server:

```bash
npx serve .
```
